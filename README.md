# Website for custcodian.dev

Yep, it's a website.

## Contributing

Feel free to send PRs, I'm enforcing DCO at the moment.

## Hot-Reloading

You will need https://astral.sh/uv installed.

```shell
uv run mkdocs serve
```

## Building

You will need https://astral.sh/uv installed.

```shell
uv run mkdocs build --site-dir _site
```
